# Vizsgaremek
A SzakiPark egy web alapú parkolónyílvánrató rendszer, amely az iskola előtti parkolókat kezeli. A projektünk egy szerveroldali backendből, egy webes frontendől,
és egy mobilalkalmazásból áll.
A projekt főbb részei:
Backend: MySQL adatbázis, Node.js + Express szerver
Frontend: Angular webalkalmazás
MobilApplikáció: Expo (React Native) mobilalkalmazás, amely egyformán futtatható Android és iOS rendszeren.
