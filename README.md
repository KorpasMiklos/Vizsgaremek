# Vizsgaremek

A **SzakiPark** egy web alapú parkolónyilvántartó rendszer, amely az iskola előtti parkolókat kezeli.  
A projekt egy szerveroldali backendből, egy webes frontendből és egy mobilalkalmazásból áll.

## A projekt célja

A rendszer célja, hogy egyszerűen kezelhető módon nyilvántartsa a parkolóhelyeket, és megkönnyítse azok kezelését az iskola környezetében.

## A projekt főbb részei

- **Backend:** MySQL adatbázis, Node.js + Express szerver
- **Frontend:** Angular webalkalmazás
- **Mobilalkalmazás:** Expo (React Native) mobilalkalmazás, amely Android és iOS rendszereken is fut

## Használt technológiák

### Backend
- Node.js
- Express
- MySQL

### Frontend
- Angular

### Mobilalkalmazás
- React Native
- Expo
